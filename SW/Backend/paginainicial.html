Onde está? Ônibus Fortaleza

Mostra localização aproximada dos ônibus e pontos de parada monitorados pelos painéis inteligentes

Este html recebe os dados de simulação da movimentação das linhas 020 (Campus do Pici Interno) e 075 (Campus do Pici Unifor), para prova de conceito. O painel inteligente de parada de ônibus está sendo simulado na parada da UFC que se localiza na Av. Humberto Monte. Ele é um hardware real, com um sensor de temperatura e umidade, coleta e envia dados reais via tecnologia LoRa para o gateway, que utiliza a tecnologia MQTT para enviar os dados até aqui.
